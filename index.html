Git

  • git status: checks what branch we are currently on
  • git pull origin main: pulls the latest version of the main branch from git
  • git checkout -b <branch-name>: creates a new branch named <branch-name> and switches to it - without the -b flag, it just switches to branch branch-name
  • git add -A: Adds all files in the current branch to the staging area
  • git commit -m "<message>": commits the changes to the repository - -m is the switch for commit message
